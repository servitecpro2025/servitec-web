# Servitec-web

Proyecto React + Vite + Tailwind + Framer Motion preparado para deploy en Vercel/Netlify.

## Pasos rápidos

1. Instalar dependencias:
```bash
npm install
```

2. Levantar en desarrollo:
```bash
npm run dev
```

3. Build para producción:
```bash
npm run build
npm run preview
```

4. Despliegue en Vercel:
- Subir este repositorio a GitHub y luego importarlo en Vercel (framework: Vite).

Logo fuente: `public/logo-servitec.jpg`
